# Perfit - Perfect Fit

---

온라인으로 옷을 구매할 때 사이즈가 맞지않아 환불하거나 교체를 해야하는 상황이 일어난다. 이런 문제점의 솔루션으로 퍼핏을 제작했다. 퍼핏은 나의 전신사진 만으로 신체 치수를 알아내고, 옷의 치수와 나의 신체 치수를 비교해준다.
