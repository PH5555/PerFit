## 📖 About The Project

온라인으로 옷을 구매할 때 사이즈가 맞지않아 환불하거나 교체를 해야하는 상황의 솔루션 앱.

![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/rainbow.png)

## 주요 기능

1. 사용자의 신체 치수 인식 및 모델 생성 기능

2. 캡처된 화면에서의 치수표 텍스트 인식 및 분석 기능

3. 가상 옷 피팅 기능

4. 피팅 헤드 기능

## 역할

- 안드로이드 개발
- 기획

## 사용 기술 스택

- JAVA & Android
- C++ & openCV
- PoseNet API
